Aide pour l'utilisation de git (a completer) :

Ajouter un fichier :
--------------------

git pull (pour verifier que vos fichiers sont bien a jour)
git add fichier.extention
git commit -am "commentaire de modification" (explicite !)
git push

Supprimer un ficher :
---------------------

git pull (pour verifier que vos fichiers sont bien a jour)
git rm fichier.extention
git commit -am "commentaire de modification" (explicite !)
git push

Les Branches :
--------------

On va se servir de branche pour eviter de se marcher dessus pendant le developpement du projet. En gros c'est plusieurs gros dossiers que l'on pourra fusionner une fois finis.

Pour les commandes les plus importantes :
git branch (voir toutes les branches dispo, celle avec une etoile est celle dans laquelle vous vous trouvez).
git branche nom_branche (pour creer une nouvelle branche pour votre groupe (ne pas creer de branche inutilement !).
gi checkout nom_branche (pour vous deplacer dans la branche nom_branche).

lien vers la partie 'branches' du tuto : http://fr.openclassrooms.com/informatique/cours/gerez-vos-codes-source-avec-git/travailler-avec-des-branches

le fichier .gitignore :
-----------------------

Il est a la racine et il ne faut pas le supprimer ! Son but est d'eviter la synchronisation des fichiers temporaires inutiles et autres.
Vous pouvez quand meme y ajouter un fichier a ne pas synchroniser (Chaque fichier sur une ligne.).

Les git commit :
----------------

Pour voir tout les commentaires et surtout les dernieres modifications apportee sans pour autant aller sur le site internet ou l'application smartphone vous pouvez utiliser la commande :
git log

Attention !
-----------

- On ne peux pas add un dossier vide.
- Ne pas mettre de noms ou de commentaires random (pas tres credible a la soutenance si on veux faire valoir notre utilisatio de github).
- Faites des 'git pull' regulierement pour garder vos fichers a jour !

Pour avoir plus d'infos repportez-vous au tuto : http://fr.openclassrooms.com/informatique/cours/gerez-vos-codes-source-avec-git
ou utiliser la commande 'git --help'
